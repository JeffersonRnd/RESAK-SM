# 🛒 STORE MANAGER

## 📌 Descripción
**STORE MANAGER** es una plataforma web desarrollada como "prototipo" para el minimarket *Resak*, orientada a mejorar la gestión de inventarios, ventas y la interacción con los clientes mediante una solución digital moderna, accesible y eficiente.

---

## 🚨 Problema
El minimarket presenta las siguientes dificultades:
- Falta de control eficiente del stock
- Registro desorganizado de ventas
- Dificultad para actualizar productos y promociones
- Ausencia de presencia digital para clientes

---

## 🎯 Objetivo
Desarrollar una plataforma web que permita:
- Gestionar inventarios en tiempo real
- Registrar y analizar ventas
- Mejorar la experiencia del cliente y del administrador

---

## 👥 Usuarios
- **Clientes:** Navegan productos y simulan compras  
- **Administradores / Empleados:** Gestionan inventario, ventas, promociones y reportes  

---

## ⚙️ Funcionalidades

### 🛍️ Cliente
- Visualización de productos
- Búsqueda por nombre o categoría
- Carrito de compras (simulación)
- Interfaz amigable

### 🧑‍💼 Administrador / Empleado
- Gestión de productos (crear, editar, eliminar)
- Control de stock en tiempo real
- Generación de reportes de ventas
- Gestión de promociones

### 📊 Sistema
- Dashboard con gráficos de ventas
- Diseño responsive (adaptable a móviles)
- Accesibilidad web (WCAG)

---

## 🧠 Características
- Interfaz intuitiva y fácil de usar  
- Soporte para múltiples roles  
- Optimización de procesos del negocio  
- Mejora en la toma de decisiones  

---

## 🛠️ Tecnologías
- HTML  
- CSS  
- JavaScript  
- Bootstrap  
- Chart.js  
- GitHub  

---

## 🚀 Alcance (MVP)
El sistema implementa un **Producto Mínimo Viable (MVP)** que incluye las funcionalidades esenciales para validar la solución y resolver los problemas principales del negocio.

---

## 📈 Impacto
- Mejora en la gestión del inventario  
- Reducción de errores operativos  
- Incremento en ventas  
- Mayor satisfacción del cliente  
- Digitalización del minimarket
